Detailed Description

template<typename T, class Comp = HeapHelper<Var, T>>
class saturnin::Heap< T, Comp >

This class represent a Heap. A data structure that allows to add different integer according to a key, and an easy access to the minimum key present in this heap. In order to work, if n integer must be present in the heap must range from 0 to n-1 and be inserted in order
