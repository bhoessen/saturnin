Detailed Description

The PoolList class is a class that provide access to an allocator of clauses of the same size. And whenever an object isn't used anymore, it is going back to the pool. When a new object is requested, we first provide element that are available in the pool. If we ran out of available object, new clauses will be created, without modifying the memory location of other clauses
